## 02. 입출력 예시
### 입력
첫째 줄에 N, M, K가 주어진다.

둘째 줄부터 M개의 줄에 파이어볼의 정보가 한 줄에 하나씩 주어진다. 파이어볼의 정보는 다섯 정수 ri, ci, mi, si, di로 이루어져 있다.

서로 다른 두 파이어볼의 위치가 같은 경우는 입력으로 주어지지 않는다.

### 출력
마법사 상어가 이동을 K번 명령한 후, 남아있는 파이어볼 질량의 합을 출력한다.

## 03. 제한 사항
* 4 ≤ N ≤ 50
* 0 ≤ M ≤ N2
* 1 ≤ K ≤ 1,000
* 1 ≤ ri, ci ≤ N
* 1 ≤ mi ≤ 1,000
* 1 ≤ si ≤ 1,000
* 0 ≤ di ≤ 7
